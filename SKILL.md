# qiuyuan-skill · 球员/教练思维蒸馏系统

> 「足球是空间的博弈，谁制造空间、谁利用空间，谁就赢。」

---

## 核心理念

qiuyuan-skill 不是复制一个球员，是**提炼他踢球的思维操作系统**。

人是多面性的——8路研究不是为了得到8份独立报告，而是为了从不同角度照亮同一个人，最终提炼出**几条连贯的因果链**，而不是一堆孤立的事实。

一个球员/教练的 Skill 是一套可运行的足球智能：
- 他用什么**战术模型**看待比赛？（镜片）
- 他在**具体场景**下怎么决策？（即时反应）
- 他的**身体条件**决定了他能做什么选择？（极限边界）
- 他处于什么样的**媒体和团队环境**？（生态背景）
- 他**绝对不会**做什么？（禁区行为/体系不适配）
- 什么是这个 Skill **做不到的**？（数据边界）

**关键区分**：捕捉的是 HOW they play/coach，不是 WHAT they said。

### Phase 0A: 需求澄清（直接路径）

收到明确名字后，进入交互模式，确认以下信息：

1. **身份确认**：球员还是教练？现役还是退役？
2. **聚焦方向**（可选）：战术层面 / 技术层面 / 精神层面 / 全面画像
3. **用途**：战术顾问？比赛分析？球员对比？
4. **本地素材**：有没有比赛录像、采访字幕、数据报告？直接丢进来，质量更高
5. **新建 or 更新**：检查 `.claude/skills/` 是否已有该球员的 Skill

交互格式（直接展示给用户确认）：

```
确认信息：
- 蒸馏对象：[球员/教练名]
- 身份：[球员/教练]，[现役/退役]
- 聚焦方向：全面画像（默认）/ 战术 / 技术 / 精神
- 用途：[战术顾问 / 比赛分析 / 球员对比]
- 本地素材：未提供（默认走网络搜索）

如需调整，请回复修改内容；没问题请说「开始蒸馏」。
```

用户说「开始蒸馏」或确认后 → 进入 Phase 1。

#### Agent 维度深度说明

有些维度的研究需要专项方法，单独说明：

**⑤ 媒体叙事 → 舆论高光时刻专项**

热梗/名场面/大众共识标签，是舆论对球员最高密度的认知压缩。

采集方法：
- B站弹幕（播放量最高的比赛视频）
- 虎扑热帖（年度投票帖、球员评分帖）
- 微博/小红书高赞评论区的"玩家共识"
- twitter moments / reddit soccer 热帖

解读方法：热梗是大众提炼出来的性格标签，每个梗背后都有一个被反复验证的真实特征。

```markdown
## 舆论高光时刻

| 热梗/名场面 | 来源平台 | 背后揭示的特征 |
|-----------|---------|-------------|
| [梗内容] | [平台] | [揭示的性格/踢球特征] |

大众共识标签：[舆论给这个球员贴的典型标签]
与实际的落差：[这个标签有多少是真的，有多少是误解]
```

**② 技术决策库 → 战术边界行为专项**

假摔、挑衅裁判、利用规则漏洞——这些不是"脏动作"，而是**有意识的战术决策**，是球员决策库的重要组成部分。

研究方法：
- 搜索该球员的红黄牌记录、假摔争议、裁判投诉
- 对比他在不同比赛/裁判下的行为差异
- 追踪争议事件的后续：他是否被"标记"为喜欢钻规则漏洞的球员

```markdown
## 战术边界行为

| 争议事件 | 比赛/时间 | 类型 | 当时决策逻辑 | 后续影响 |
|---------|---------|------|------------|---------|
| [事件]   | [时间]   | [类型] | [他当时在做什么] | [裁判/对手/舆论反应] |

边界决策模式：
- 规则利用倾向：[主动利用/战术需要/偶尔为之/从不]
- 风险收益比：他是否计算过被抓的代价
- 变化轨迹：[早期更激进/近期收敛/一直如此]
```

**⑥ 社交动态 → 争议事件行为模式专项**

场下争议（转会/言论/私生活）揭示的是球员在**非竞技压力**下的真实反应，这类信息无法伪造。

处理方式比争议本身更重要：
- **沉默型**：不回应，让时间冲淡（≠认错）
- **反击型**：公开反驳，争取舆论同情
- **道歉型**：认错但不过度，保留尊严
- **转移型**：避重就轻，用其他话题覆盖
- **策略型**：借争议达到某种目的

```markdown
## 争议事件行为模式

| 争议事件 | 时间 | 类型 | 他的处理方式 | 后续结果 |
|---------|------|------|------------|---------|
| [事件]   | [时间] | [类型] | [处理方式] | [更衣室/舆论/场上表现变化] |

争议处理模式总结：
- 情绪类型：[激烈型/压抑型/策略型]
- 典型反应：[私下解决/公开回击/沉默冷处理]
- 风险边界：[什么话题/情境会触发他的失控]
- 自我修复能力：[事后能否正常比赛/是否影响更衣室]
```

### Phase 1.4: 跨维度交叉阅读

> **这个阶段是整个设计的核心。8路不是终点，跨维度连接才是。**

每个 Agent 在完成自己的报告后，必须阅读其他7路的报告，寻找以下三类连接：

#### 三种跨维度连接类型

| 类型 | 定义 | 格式 |
|------|------|------|
| **印证** | 我的发现被别的维度证实了 | 「A维度的X现象 → B维度也有相同指向」 |
| **矛盾** | 别的维度揭示的东西和我的描述不一致 | 「A说X，但B的数据/行为显示Y」 |
| **涌现** | 单独看都合理，但连起来才发现的新洞察 | 「A和B单独都合理，但连起来揭示了一个更深层的逻辑」 |

#### 跨维度发现矩阵（输出文件：`references/research/09-cross-matrix.md`）

格式示例：

```markdown
## [球员名] 跨维度发现矩阵

### 发现 #1：身体 → 技术 → 空间（印证链）
- 维度：A③身体与极限 ↔ A②技术决策库 ↔ A①空间决策
- 内容：重心低+重心转移快（身体）→ 变向假动作成功率极高（技术）→ 能在密集防守中保持持球（空间）
- 证据：[具体比赛场景]
- 意义：这个技术动作不是学来的，是身体条件驱动的本能决策

### 发现 #2：媒体叙事 ↔ 体系契合（矛盾）
- 维度：A⑤媒体叙事 ↔ A④体系契合
- 内容：媒体认为他是"体系球员，离开强队就不行"（⑤），但数据显示他在低控球率比赛中的胜率反而更高（④）
- 矛盾点：舆论和实际情况相反
- 意义：这可能是他选择新球队时最应该参考的维度

### 发现 #3：社交动态 → 关键时刻（涌现）
- 维度：A⑥社交动态 → A⑦关键时刻
- 内容：他在社媒上从不说大话（⑥）→ 但在决赛/重要比赛时的表现历史极强（⑦）
- 涌现洞察：他不需要用嘴建立自信，场上表现本身就是他的表达方式
- 意义：这条连接比任何单一维度的描述更能揭示他的真实性格
```

#### 设计哲学：为什么交叉阅读比并行更重要

单一维度看到的是**数据点**，跨维度连接看到的是**因果链**：

```
并行8路输出：
"他内收"  "左脚内切"  "媒体说他冷静"
  ↓           ↓            ↓
三个独立事实，拼在一起还是散的

跨维度连接后：
他内收（①空间）
 → 变向快（②技术）
 → 因为重心低（③身体）
 → 所以他不依赖语言表达（⑥社交）
 → 在大场面反而更冷静（⑦关键时刻）
  ↓
一条连贯的因果链 = 真正的球员理解
```

**人不是8个维度的叠加，人是一条因果链上的节点。** 交叉阅读的目的就是找到那条链。

### Phase 2: 框架提炼

> 所有框架提炼必须基于跨维度发现矩阵，不能只看单一维度的报告。

#### 2.1 战术模型（5个）

来源：优先从"印证"类跨维度发现中提炼，因为有多个维度支撑，可信度更高。

操作步骤：
1. 扫描跨维度发现矩阵，列出所有有多个维度印证的候选战术特征
2. 按三重验证筛选：
   - **跨场景验证**：在≥2个不同比赛场景中出现？
   - **生成力**：能推断此球员对新场景的反应？
   - **排他性**：不是所有这个位置的球员都这样？
3. 取 top 3-5 个，按排他性排序

#### 2.3 表达DNA分析（球员说话方式）

来源：综合②技术决策库（场上决策的语言反映）+ ⑤媒体叙事（舆论给他的标签）+ ⑥社交动态（他主动说的话）+ ⑦关键时刻（高压下的语言反应）。

**为什么这个维度重要**：球员说话的方式，就是他在场上的决策方式的延伸。不了解一个球员怎么说话，就无法在对话中还原他的视角。

提取维度：

| 维度 | 问题 | 示例 |
|------|------|------|
| **词汇特征** | 说什么词？专业术语还是大白话？ | 梅西几乎不用数据术语；瓜迪奥拉张口就是空间/角度 |
| **确定性表达** | 说话果断还是留余地？"毫无疑问"vs"我不确定" | C罗永远确定；莱奥经常说"我也不知道怎么回事" |
| **第一人称使用** | 赢球说"我们"还是"我"？输球呢？ | C罗：赢球全是"我"，输球推给"球队" |
| **情绪外露程度** | 公开场合能感受到情绪吗？ | 穆里尼奥演讲情绪外露；瓜迪奥拉极度克制 |
| **幽默/讽刺** | 采访里会开玩笑吗？讽刺谁？ | 穆里尼奥讽刺记者是日常；克洛普用幽默化解紧张 |
| **话题禁区** | 哪些话题他绝对不接？ | 某些球员不评价裁判；某些球员不聊家庭 |
| **团队vs个人** | 赢了夸队友还是夸自己？输了怪谁？ | 揭示真实的领袖力和责任感 |
| **对对手的态度** | 评价对手时是尊重还是攻击？ | 梅西几乎不攻击对手；C罗经常暗示自己最强 |

**输出格式**（每项都要有具体引例）：

```markdown
## leao 表达DNA

**词汇特征**：简短、口语化、几乎没有抽象术语。不说"我的战术职责是..."，说"我就是拿到球然后想办法过掉他"。
**确定性**：中等偏弱。经常说"我不知道能不能成"然后做成了。自我怀疑型但行动果断。
**第一人称**：赢球我，输球我们。
**情绪**：场上极度外露（进球后狂吼）；场下极度克制（赛后采访面无表情简短回答）。
**幽默**：几乎没有幽默，队友爆料他从不讲笑话。
**话题禁区**：①不评价队友表现 ②不聊家庭 ③不解释转会传闻。
**团队vs个人**：极端个人主义——进球后从不第一时间找队友庆祝。
**对对手**：从不攻击对手。被问到"对手哪里强"，他会说"我不知道他们强在哪，反正我们赢了"。
```

**与nuwa-skill的区别**：nuwa-skill的"表达DNA"是通用维度；球员版要去掉"引用习惯"（球员几乎不引经据典），增加"团队vs个人"和"对对手态度"这两个足球领域特有的维度。

## 品味守则

| 原则 | 说明 |
|------|------|
| **比赛 > 数据** | 一场完整比赛比5个集锦更能揭示思维 |
| **矛盾 > 完美** | 球员的内在矛盾（攻强守弱/领袖vs独狼）最能揭示真实 |
| **变化 > 固定** | 巅峰期和早期踢法有差异，记录变化才有价值 |
| **生态 > 孤立** | 球员不是机器，媒体环境和团队状态直接影响场上表现 |
| **连接 > 堆砌** | 8路研究的价值在于发现连接，不在于收集更多孤立事实 |

**绝不做的事**：
- 编造比赛细节（没看过的比赛不捏造）
- 把数据当结论（xG高不等于踢得好，要看场景）
- 忽略负面评价（这个球员被批评的地方同样重要）
- 在信息不足时强行生成完整 Skill
- 把媒体报道当真相（舆论和现实往往有差距）
- 把8路报告简单拼凑（必须做跨维度连接后才能进入提炼阶段）
