# qiuyuan-skill

球员/教练思维蒸馏系统。输入球员或教练姓名，自动深度调研→战术框架提炼→生成可运行的球员 Skill。

---

## 是什么

qiuyuan-skill 能够将球员/教练的思维方式蒸馏成**可运行的足球智能**——输入任意球员名，自动采集其战术哲学、技术风格、比赛决策模式，生成可直接用于战术分析的 Skill。

不是复刻集锦，是提炼**踢球的思维操作系统**。

---

## 8路信息采集

球员是社会中人，也是场上决策者。采集分三层：

| 层次 | Agent | 研究什么 |
|------|-------|---------|
| **踢球层** | ①空间决策、②技术决策库、③身体与极限、④体系契合 | 他怎么踢的？ |
| **生态层** | ⑤媒体叙事、⑥社交动态 | 他现在处于什么环境和状态？ |
| **背景层** | ⑦关键时刻、⑧生涯轨迹 | 他为什么会这样踢？ |

### 核心4路

| Agent | 核心问题 |
|-------|---------|
| ① 空间决策 | 他在无球时在哪里？进攻展开时他制造还是利用空间？ |
| ② 技术决策库 | 他接到球第一反应是什么？技术动作是决策的延伸？ |
| ③ 身体与极限 | 他的身体条件决定了他能做什么选择？ |
| ④ 体系契合 | 什么体系让他更强，什么让他消失？ |

### 生态2路（新增）

| Agent | 核心问题 | 为什么重要 |
|-------|---------|---------|
| ⑤ 媒体叙事 | 俱乐部/教练怎么看他？外部期待如何影响场上角色？ | 直接影响更衣室状态和场上压力 |
| ⑥ 社交动态 | 他怎么表达自己、处理关系？ | 揭示真实性格，与踢球风格互为印证 |
