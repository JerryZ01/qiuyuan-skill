# qiuyuan-skill

球员/教练思维蒸馏系统。输入球员或教练姓名，自动深度调研→战术框架提炼→生成可运行的球员 Skill。

---

## 是什么

qiuyuan-skill 能够将球员/教练的思维方式蒸馏成**可运行的足球智能**——输入任意球员名，自动采集其战术哲学、技术风格、比赛决策模式，生成可直接用于战术分析的 Skill。

不是复刻集锦，是提炼**踢球的思维操作系统**。

---

## 核心能力

### 8路并行信息采集

| Agent | 研究维度 | 核心问题 |
|-------|---------|---------|
| 战术哲学 | 体系定位、位置职责、攻防偏好 | 这个球员在体系里干什么？ |
| 技术动作库 | 盘带习惯、传球脚法、射门选择 | 他怎么踢？标志动作是什么？ |
| 身体与心理 | 速度耐力、大场面、心态 | 逆境怎么处理？大赛软脚吗？ |
| 生涯轨迹 | 成长路径、巅峰期、转折点 | 在哪里踢出来？什么环境激发最强状态？ |
| 团队化学 | 领袖风格、体系契合、队友配合 | 更衣室老大还是场上身先士卒？ |
| 对手博弈 | 强队/弱队表现差异、大赛数据 | 遇强愈强还是欺软怕硬？ |
| 公开表达 | 采访风格、社交媒体、领袖喊话 | 他怎么说话？情商高低？ |
| 数据画像 | 关键统计、风格标签、能力雷达 | xG、传球成功率、场均跑动 |

### 触发方式

```
蒸馏梅西
分析哈兰德的踢法
做个莱奥的战术视角
用姆巴佩的视角分析这场比赛
```

## 安装

```bash
# 方法1：直接从 GitHub 安装
npx skills add JerryZ01/qiuyuan-skill

# 方法2：clone 后本地安装
git clone https://github.com/JerryZ01/qiuyuan-skill.git
# 将内容复制到 ~/.claude/skills/qiuyuan.skill/
```

安装后在 Claude Code 中直接说自然语言触发。

## 信息采集

足球领域信息主要来自视频素材，配套工具链：

```
scripts/
├── download_subtitles.sh   # YouTube 视频字幕下载
├── srt_to_transcript.py    # 字幕清洗（SRT → 纯文本）
├── collect_sources.py      # 球员视频/数据元数据采集
└── MANUAL_COLLECTION.md    # 完整采集操作指南
```

> ⚠️ WSL2 下部分平台存在网络限制，建议参考 `scripts/MANUAL_COLLECTION.md` 的跨平台采集流程。

## 生成结果

蒸馏完成后生成独立 Skill 目录：

```
~/.claude/skills/梅西-perspective/
├── SKILL.md          ← 可运行的球员思维操作系统
└── references/
    ├── research/     ← 8路Agent采集结果
    └── sources/      ← 一手素材（采访字幕/数据）
```

### SKILL.md 结构

```markdown
## ⚡ 角色扮演规则          # 沉浸式球员视角
## 回答工作流              # 比赛分析 → 研究 → 回答
## 战术模型（5个）          # 核心踢球逻辑
## 场景决策库              # 具体比赛场景即时反应
## 表达DNA                 # 采访风格、领袖喊话方式
## 禁区行为               # 绝对不会做的事
## 诚实边界               # 数据捕捉不到什么
```
